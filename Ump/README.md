## 📖 Описание

**Ump** - современная музыкальная платформа для прослушивания треков и просмотра видео контента. Проект демонстрирует сложную интерактивную веб-платформу с системой пользователей и мультимедиа контентом.

## ✨ Особенности

- 🎵 **Музыкальная платформа** - прослушивание треков онлайн
- 🎬 **Видео контент** - просмотр музыкальных клипов  
- 👤 **Система аккаунтов** - регистрация и авторизация
- 🔍 **Поиск и фильтрация** - удобный поиск контента
- 💬 **Комментарии** - система комментариев к трекам
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- ⚡ **Gulp сборка** - современная система сборки

## 🚀 Установка и запуск

### Предварительные требования

- Node.js (версия 16 или выше)
- npm

### Установка

1. Перейдите в папку проекта:
```bash
cd Ump/gulp-maxgraph
```

2. Установите зависимости:
```bash
npm install
```

### Команды

```bash
# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Проверка стилей
npm run stylelint

# Исправление ошибок стилей
npm run style-fix

# Проверка EditorConfig
npm run code

# Создание архива
npm run zip
```

## 🎨 Страницы и функционал

### 📄 Страницы:
- 🏠 **Главная** (`index.html`) - обзор треков и плейлистов
- 🔐 **Вход/Регистрация** (`main-login.html`) - авторизация пользователей
- 👤 **Личный кабинет** (`main-account.html`) - управление профилем
- 🎬 **Видео** (`main-video.html`) - просмотр видео контента

### 🧩 Компоненты:
- 🎵 **Плеер** - воспроизведение треков
- 🔍 **Поиск** - поиск по трекам и исполнителям
- 📊 **Чарты** - топовые треки
- 💬 **Комментарии** - обсуждение треков
- 📱 **Адаптивное меню** - мобильная навигация
